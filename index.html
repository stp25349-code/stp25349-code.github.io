ข้อมูลโปรไฟล์ | Data Profile

ชื่อ: Oakslayers

อายุ: 18 ปี

ในโลกดิจิทัลที่เชื่อมต่อกัน ผมคือผู้สร้างสรรค์ที่หลงใหลในศิลปะการโค้ดดิ้งและการออกแบบเกม ผมเชื่อว่าเกมคือมากกว่าแค่ความบันเทิง มันคือโลกที่เราสามารถสร้างสรรค์และเล่าเรื่องราวได้อย่างไร้ขีดจำกัด

ความสนใจ | Core Interests

ผมมีความสนใจเป็นอย่างมากในโลกของเกมดิจิทัล ไม่ว่าจะเป็นเกมแนว RPG ที่มีเรื่องราวลึกซึ้ง หรือเกมแนว Strategy ที่ต้องใช้ความคิดอย่างหนัก

  • การออกแบบตัวละครและโลกในเกม
  • การเขียนโค้ดเพื่อสร้างกลไกเกมที่ซับซ้อน
  • การสำรวจเทคโนโลยีใหม่ๆ ในวงการเกม เช่น VR และ AR

การสร้างเกม | Game Dev Logs

การสร้างเกมเป็นสิ่งที่ผมหลงใหลที่สุด ผมสนุกกับการเปลี่ยนแนวคิดให้กลายเป็นโลกเสมือนจริงที่ทุกคนสามารถเข้ามาสัมผัสได้

เกมแนว Sci-Fi RPG

ข้อมูลการศึกษา | Academic File

ผมเป็นนักเรียนที่ โรงเรียนสุรธรรมพิทักษ์

ที่นี่ ผมได้พัฒนาทักษะด้านเทคโนโลยีและคอมพิวเตอร์ผ่านโครงการต่างๆ และนำความรู้ไปประยุกต์ใช้ในการสร้างเกม

งานอดิเรก | Leisure Protocol

งานอดิเรกหลักของผมคือการเล่นเกม ไม่ว่าจะเป็นการค้นหา Easter Eggs ในเกมใหม่ๆ หรือการฝึกฝนเพื่อพิชิตความท้าทายในเกมที่ชื่นชอบ

  • แนวเกมที่ชอบ: RPG, FPS, Simulation
  • แพลตฟอร์ม: PC, Console (PlayStation/Xbox)
  • เกมที่กำลังเล่น: Cyberpunk 2077, The Witcher 3
